# 웹 공유 QA 체크리스트

환경: Supabase 마이그레이션 `0077_meeting_share_links.sql` 적용 후, 앱에 `EXPO_PUBLIC_MEETING_SHARE_WEB_URL`(끝 슬래시 없이)과 웹에 동일 Supabase URL·Anon 키 설정.

## OPEN 모임

1. 호스트가 앱에서 **웹 공유** → 시트에 URL 포함되는지 확인.
2. 비로그인 브라우저에서 `/s/{token}` 열기 → 제목·후보·투표 UI 표시.
3. 표시 이름 입력 후 **참여하기** → 앱 모임 상세에서 참여자 수·투표 반영.
4. 같은 브라우저에서 **투표 저장** → 집계 변경 확인.
5. 다른 브라우저에서 동일 링크로 새 참여 → 정원 초과 시 에러 메시지.

## HOST_APPROVAL 모임

1. 웹에서 **참가 신청**(필요 시 메시지) → 앱에서 대기 신청 표시.
2. 호스트 승인 → `ginitweb_` 참여자가 일반 참여자처럼 집계에 반영되는지.
3. 승인 전 웹에서 **투표 저장** → `joinRequests` 행만 갱신되는지.

## 토큰

1. 잘못된 토큰·만료(365일 후)·폐가된 링크 → 안내 문구.

## 앱 UI

1. 참여자 줄에서 웹 게스트 닉 표시 및 탭 시 「웹 참여자」 안내.
